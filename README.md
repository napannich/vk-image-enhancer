# VK Image Enhancer

ML-модель для улучшения изображений в браузере.

## Описание
Веб-приложение для автоматического улучшения качества изображений (яркость, контрастность, цветность) с использованием ML-модели, работающей непосредственно в браузере пользователя через TensorFlow.js.

## Стек
- TensorFlow.js — inference в браузере
- Web Workers — асинхронная обработка
- Canvas API — манипуляция изображениями
- TypeScript
- GitHub Pages — хостинг

## Требования
- Работоспособность во всех современных браузерах
- Код до 10 МБ
- Обработка до 15 Мпикс
- Среднее время обработки — 5 с
- Поддержка: JPG, PNG, HEIC, BMP
